# Requisitos de Hardware y Software
- **Servidor/Hosting**: Vercel (Free/Pro), Supabase administrado.
- **Desarrollo local**: 8–16 GB RAM, Windows 10/11 o Linux, Node.js LTS, Git, VS Code, navegador actualizado.
- **Software**: Node.js LTS, npm/pnpm, Supabase CLI (opcional), Vercel CLI, GitHub.
- **Rendimiento objetivo**: <300 ms p95 en vistas públicas; disponibilidad 99.5% mensual.
- **Seguridad**: HTTPS, CSP, cookies seguras, RLS en BD, backups automáticos.
